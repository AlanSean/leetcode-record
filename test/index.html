Given a linked list, remove the n-th node from the end of list and return its head 给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。 example: 示例： Given linked list: 1->2->3->4->5, and n = 2. 给定一个链表: 1->2->3->4->5, 和 n = 2. After removing the second node from the end, the linked list becomes 1->2->3->5. 当删除了倒数第二个节点后，链表变为 1->2->3->5.
